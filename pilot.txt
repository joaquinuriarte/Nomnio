The fileObject will be the name of any object that you assign the open variable to. The offset n variable is required and denotes the position of the read/write pointer within the file that you are “seeking”. The whence argument is optional and defaults to 0, which means absolute file positioning, other values are 1 which means seek relative to the current position and 2 means seek relative to the file's end.
